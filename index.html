.js-lessness

A ponderous collection of ECMAScript-free user interface elements. The author of this collection is far from an enemy of Javascript, but recognizes there is a time and a place for everything.

Helptext-oriented popup #1

For use when you want deliberate de/activation via :focus (clicking or tabbing). The popup text is inherently difficult to select (selection removes focus, and closes the popup). De-focusing upon second click in the same location is accomplished by using CSS to move a transparent div into place upon focus.
